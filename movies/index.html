顽主
2019-03-09 / 北京 国产 喜剧 青春 / ★★★★☆ 推荐
看这个片子的时候感觉那个时候的人说话很随便（很有哲理），看看这部片子的时间1988，还差一年……这里面有两个很正面的角色一个是老师，一个是教授，以前认为老师和教授都是很神圣的职业，但是这里面的却不是什么正面的角色。倒是感觉里面的三个小混混活的很潇洒，真的感觉人生这个东西真不能随便讲，当主流价值观碾压小众的价值观的时候，那就是正义吗？有个词叫做“普师价值观”，以这个价值观为本体向外生长出一些个人的价值观，这样才符合正常事物的发展。之所以看这部电影主要是因为这句话：实在不由自主就当自己是在老山前线一人坚守阵地，守得住光荣，守不住也光荣！感觉说的挺在理的。感觉那时候拍的电影挺好的，现在的电影自己看的不多，做不了太多的评价。
白蛇：缘起
2019-03-05 / 动画 古装 唯美 / ★★★★☆ 推荐
作为一部动漫电影，里面的画面做的很漂亮。故事的情节虽然说不上太好，但对于少年儿童来说，应该还行吧。想想自己也已经过了看这种类型的动漫的年龄了。这里面的爱情故事可能只适合于那些仍然处于无忧无虑的“孩子”身上吧。以前我也是很喜欢这种美丽的爱情故事，但是当自己长大了之后，也开始发现生活并不是童话，也开始相信“童话里都是骗人的”这句话了。随着自己年龄的增长，以前没有意识到的事情现在开始慢慢的有了些察觉。相比于这里面的纯洁的爱情，我更喜欢《十日谈》中那些“真实”爱情。可能真的是因为年龄的原因吧。看这种电影主要还是为了消遣时间吧……

小妇人 / Little Women
2019-02-19 / 女性 名著改编 美国电影 / ★★★★★ 力荐
里面的演员真的很漂亮。艾米很小的时候就说，她们将来都会长大，现在她们应该想清楚自己想要什么。西方文化从小 就教孩子要学会独立，知道自己想要什么。而在东方，我们的父母总是百般的“爱护”自己的孩子，可是这种爱又有多少是真正无偿的呢？又有多少父母养孩子是为了“养儿防老”？有一点不服从父母的心意，就担心父母说自己翅膀硬了，毕竟是儒家文化中培养出来的，说自己一点也不在乎别人怎么说，可是从小在这种文化中成长，就算别人不说自己心里好像也过意不去。虽然说自己的思想是独立的，可是在金钱上还是要靠家里。以前给父母要钱感觉理所应当，可是现在真的很难张开嘴再要。后悔当时没有申请助学贷款。乔宁愿卖掉自己的头发也不向姑妈借钱，这种骨气并不是中国人独有的。不得不说乔的个性真的很好，beth因为“善良”害了自己。性格决定命运。
十日谈 / Il Decameron
2019-02-18 / 十日谈 文艺复兴 / ★★★☆☆ 还行
透过这部电影我了解到文艺复兴是一场解放人性的运动，而“性”是人性中很重要的一部分。以现在人的观点来看那个时候的欧洲人民也是愚昧至极，书中描写的关于人的贪婪，欲望，等级，有时也感到自己也是深陷其中。教会对人的欲望的控制，和教会管理人员的愚昧，才是人类社会发展的巨大阻力。书中的两段爱情故事对我的印象也比较深刻：第一个是一个富家子弟喜欢上了一个算是门当户对的小姐，女方的父母知道了这件事后很开心，认为这个男子人品还不错家里也有钱，很乐意让女儿嫁给他，男方也很乐意娶那个小姐，小说家会把这写成一段美好的姻缘。另外一个是，一个搬运工和一个贵族小姐相爱，两个人彼此喜欢。女方的哥哥们知道后很生气，设计把那个搬运工给杀害了。爱情价虽高，但并不是每个人……通过这部电影我看到了很多人性的阴暗面。

长恨歌
2019-02-13 / 长恨歌 电影 中国 / ★★★★☆ 推荐
没看过原著，所以不好评论原著和电影的差异。有人把王安忆和张爱玲作比较，虽然自己没有读过两位女作家的很多作品，但是仅从《长恨歌》和《色戒》这两部电影来看感觉风格有些相似之处。女主王琦瑶先后爱上的三个人都先后离自己而去，却始终不肯接受一直在自己身边照顾自己的程先生。我不知道为什么会这样，我自以为是王琦瑶不肯原谅自己。中国有句古话：夫妻本是同林鸟，大难临头各自飞。想想王琦瑶的一生不就是这样吗？但是程先生对王琦瑶的爱恋，又让我有了反驳的依据。并不是什么东西都有标准答案，在自己年轻力壮的时候就应该早吃些苦。一座城市并不会老去，因为每天都有人奔向灿烂的青春。虽然说影片的结尾并不是大圆满，好像这样的结局更能给一些年轻人奋斗的理由。

“饮食男女，人之大欲焉。”总感觉饮食男女不是一个好的词，但是电影的最后，里面的角色大部分都成为了“饮食男女”。既然是人之大欲，为什么第一感觉总是觉得这是一个贬义词？这里面对我印象最深的是电影刚开始的时候，朱家倩和她的前男友在鱼水之欢的时候，镜头突然切换到了老朱（她的父亲）往一只煮好的鸡里面吹气。也许那个时候的朱家倩和她的男友就像那只煮好的鸡一样只是一副臭皮囊。与之相反的是大姐朱家珍，她是一个基督徒，在某种意义上是臭皮囊的对立面。可是当夜晚有两只发情的猫在叫的时候，在那学习的朱家珍却不能够忍受，而另一个老朱却能够沉下心来认真写自己的食谱。只有自己经历过才能够真正的把一些事情看得很淡吧。大姐之所以这个样子也与她从小担负起家里的责任有关吧。我猜测老朱之所以不让二姐当厨师，可能和她的母亲有关。

城南旧事
2019-01-29 / 名著改编 经典 / ★★★★☆ 推荐
里面的那个小女孩确实很可爱也很天真，从小生活在一个中产阶级的家庭中，以一个大小姐的身份来看待人间的悲欢离合。这部电影到底想讲述的是什么？《送别》这首歌贯穿整部电影，看着秀珍，偷东西的哥哥，爸爸，家里的保姆相继离自己而去。这让我想起了苏轼的那句“人有悲欢离合，月有阴晴圆缺，此事古难全。”感觉这部电影有点红楼梦的意味。童年是美好的，可是经历过这些事情之后，英子也会比同龄人更加的成熟吧。
围城
2019-01-22 / 电影 经典 名著改编 围城 / ★★★★★ 力荐
“围在城里的人想逃出来，城外的人想冲进去，对婚姻也罢，职业也罢，人生的愿望大都如此。”钱钟书先生写的这部小说虽然我没有读过原著，但是看过这部电视剧之后发现，里面的很多句子都是金句，细细琢磨亦有不可言语之妙。孙柔嘉这个女人刚开始表现的很单纯（白莲婊），千方百计想着嫁给方鸿渐结婚，结婚后结婚前完全是两个人，这种女人以后要远离。还有赵辛楣本来是情敌，但是后来和方成为了患难之交，原因就是苏文纨嫁给了曹元朗，我只能感叹男人们和女人之间的关系是那么的微妙。方鸿渐和唐晓芙是相互喜欢，最后却因为误会没有走到一起。“人生就像一席华丽的长跑，贴近了看发现上面爬满了虱子。”方鸿渐刚开始凭借自己的才华，有很多有钱的小姐都喜欢他。结尾的时候却流落街头只想找个地方安心的睡觉。是方鸿渐无能，还是孙柔嘉……
童年的许诺 / La promesse de l'aube
2019-01-06 / 传记 成长 剧情 亲情 / ★★★★★ 力荐
以前看过一篇文章写的是为什么妈宝男更容易成功，看过这部电影之后自己对这个道理又有了一个新的理解。电影中的男主从小就被母亲告知说：你将来会成为大文豪，会成为法国公使。尽管身为犹太人的她们在当时遭到很多人的冷眼，但是性格要强的母亲仍然依靠自己的智慧让自己的儿子接受上流社会应该有的教育。电影中的母亲也不是完美的化身，她看不起画家，喜欢在自己的周圈炫耀自己的儿子。但是母亲的要强和望子成龙的希望，是男主最后能够成功的关键因素。每一个成功人士的背后都有一个要强的女人吧，当自己的子女遭受到困难（屈辱）时，自己是不是应该舍弃那些所谓的道德荣辱，为孩子争取到发展的机会，告诉他们将来应该成为什么样的人？母亲告诉儿子“士可杀不可辱”，说：一个男人只能为女人，荣耀和国家来给被人打架。即使儿子把上级的老婆睡了（未完）
后来 / Afterwards
2019-01-03 / 生死 剧情 法国电影 / ★★★★☆ 推荐
面对死亡这个问题，大多数人会选择逃避。看过这部电影之后，我明白了，活着就重要的事情就是珍惜当下的每时每刻。生死有命，富贵在天。虽然说不太符合当下的主流价值观，但是里面蕴含着深刻的哲理，也许只有当自己经历过之后才能够明白吧。心学大师王阳明提出”知行合一“的观点，要是一个人真的能够做到知行合一，那么这个人是永生的，死亡对他们来说就是生命的另外一种存在形式。就像陈果老师说的那样，有的人站的足够高，他们能够看到死亡后面的东西，所以他们并不畏惧死亡。生命是短暂的，死亡却能够永生。在短暂的生命中并不是每一个人都真正的活过。
